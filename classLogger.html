Detailed Description

A singleton class for logging messages with different log levels.

The Logger class provides a thread-safe mechanism to log messages to a file. It ensures that only one instance of the logger exists throughout the application.

Logs a message with a specified log level to the log file.

This function writes a timestamped log message to the log file if it is open. The log message includes the current date and time, the log level, and the provided message. Thread safety is ensured using a mutex.

Parameters
messageThe message to be logged.
levelThe log level of the message (e.g., INFO, WARNING, ERROR).
Note
The function uses platform-specific methods to safely convert the current time to a std::tm structure.
The log file must be open for the message to be written.
