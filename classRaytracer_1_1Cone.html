Constructs a Cone object with the specified color and transformation matrix.

Parameters
cThe color of the cone, including RGB values, metallic property, and transparency.
matrixA 4x4 transformation matrix that defines the cone's position, size, and orientation.

The constructor initializes the cone's apex, radius, height, and color based on the provided matrix and color. It also determines the cone's axis of orientation ('x', 'y', or 'z') based on the value of the matrix's (2, 2) element. Additionally, it applies the transformation matrix to the cone.

Determines if a ray intersects with the cone in local space.

Parameters
rayOriginThe origin of the ray in local coordinates.
rayDirThe direction of the ray in local coordinates.
tOutOutput parameter to store the distance to the intersection point, if any.
normalOutOutput parameter to store the surface normal at the intersection point, if any.
Returns
true If the ray intersects the cone or its base disk.
false If the ray does not intersect the cone or its base disk.

This function calculates the intersection of a ray with a cone defined by its apex, height, radius, and orientation axis ('x', 'y', or 'z'). It considers both the conical surface and the base disk of the cone. If an intersection is found, the function updates tOut with the distance to the intersection point and normalOut with the surface normal at that point.

The function handles the following:

  • Intersection with the conical surface.
  • Intersection with the base disk.
  • Ensures the intersection point lies within the valid height range of the cone.

Special cases:

  • If the ray is parallel to the cone's axis or base disk, the function handles these cases appropriately to avoid division by zero.
  • If the intersection point lies outside the cone's height or base radius, it is discarded.
