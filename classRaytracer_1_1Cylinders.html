Determines if a ray intersects with the cylinder in local space.

Parameters
originThe origin of the ray in local coordinates.
dirThe direction of the ray in local coordinates.
tOutput parameter to store the distance to the intersection point, if any.
nOutput parameter to store the surface normal at the intersection point, if any.
Returns
true If the ray intersects the cylinder or its caps.
false If the ray does not intersect the cylinder or its caps.

This function calculates the intersection of a ray with a cylinder defined by its position, radius, height, and orientation axis ('X', 'Y', or 'Z'). It considers both the cylindrical surface and the top and bottom caps of the cylinder. If an intersection is found, the function updates t with the distance to the intersection point and n with the surface normal at that point.

The function handles the following:

  • Intersection with the cylindrical surface.
  • Intersection with the top and bottom caps.
  • Ensures the intersection point lies within the valid height range of the cylinder.

Special cases:

  • If the ray is parallel to the cylinder's axis or caps, the function handles these cases appropriately to avoid division by zero.
  • If the intersection point lies outside the cylinder's height or radius, it is discarded.
