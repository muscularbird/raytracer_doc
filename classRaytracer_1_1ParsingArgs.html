Detailed Description

A class to parse and manage command-line arguments for the raytracer application.

This class is responsible for extracting and storing information from the command-line arguments, such as the scene file name, output file name, and whether the application should run in graphical mode.

Raytracer::ParsingArgs::ParsingArgs ( char **  argv)

Constructor for the ParsingArgs class.

This constructor parses the command-line arguments provided to the program and initializes the ParsingArgs object with the appropriate settings.

Parameters
argvA null-terminated array of C-style strings representing the command-line arguments.
Exceptions
std::invalid_argumentIf an unknown argument is encountered, if the scene file is not specified or does not have a .cfg extension, or if the output file is specified but does not have a .ppm extension.

The following command-line arguments are supported:

  • -g: Enables graphical mode.
  • -f <scene_file>: Specifies the scene file to be used. The file must have a .cfg extension.
  • -o <output_file>: Specifies the custom output file. The file must have a .ppm extension.
