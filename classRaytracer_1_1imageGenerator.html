Constructs an imageGenerator object and opens the specified file for writing.

This constructor initializes the imageGenerator object and attempts to open the specified file in output mode. If the file cannot be opened, an error is logged using the Logger instance, and an exception is thrown.

Parameters
file_nameThe name of the file to be created or opened for writing. Defaults to "output.ppm" if no file name is provided.

Raytracer::imageGenerator::~imageGenerator ( )

Destroy the imageGenerator object and ensure the file is properly closed.

This destructor closes the file stream associated with the imageGenerator object. If the file stream remains open after attempting to close it, an error is logged using the Logger instance, and an exception is thrown.

Outputs the components of a Vec3 object to an image file.

This function takes a Vec3 object, converts its x, y, and z components to a string format separated by spaces, and writes the resulting string to the image_file stream followed by a newline.
